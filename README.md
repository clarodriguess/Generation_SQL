## Atividade Prática - Modelagem e Manipulação de Dados com SQL

Este repositório contém exercícios práticos de SQL desenvolvidos durante o Bootcamp Full Stack da Generation.

O objetivo foi treinar conceitos fundamentais de banco de dados, incluindo modelagem, criação de tabelas e manipulação de dados.

---

### Tecnologias Utilizadas

- SQL
- MySQL

---

### Estrutura

Os arquivos do repositório incluem:

- Scripts de criação do banco de dados
- Scripts de criação das tabelas
- Inserção de dados
- Consultas para manipulação e análise

---

### Conceitos Praticados

- Modelagem relacional
- Integridade referencial
- CRUD (Create, Read, Update, Delete)
- Relacionamento entre tabelas
